Docker をセキュアに使うために

Aug 06, 2019

開発に Docker を使う機械が増えてきましたが使っててきになるのがセキュリティです。

そこで Docker をセキュアに使うためにも自分ようのメモも兼ねて書いておきます。

まず root 権限で docker グループにパスワードを設定します。

sudo gpasswd docker

一時的に docker を使いたいユーザーを Docker グループに参加します

newgrp docker

これで sudo を使わず Docker を使えます!
